# FFXV-Superfan

## Repository layout
- `canon/` — fixed Final Fantasy XV canon reference system (source of truth).
- `fanon/` — flexible fandom interpretation reference system (non-canon patterns).
- `stories/` — story-specific workspaces (drafting notes, project references, and planning).

Use `canon/` for stable canonical facts, `fanon/` for interpretation patterns and AU conventions, and keep each story's custom material in its own story folder.
