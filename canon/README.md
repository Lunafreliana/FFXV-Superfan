# Canon Reference System (Fixed)
This directory stores **Final Fantasy XV canon only**.
- Canon files are intended to be stable, high-confidence references.
- Story-specific brainstorming, alternate routes, and headcanon belong in top-level story folders (for example, `stories/mermaiding/`), not here.
- The `canon/` tree is designed as a fixed source of truth that story projects can consult for continuity.
## Canon structure
- `canon/index.md` — master index of all canon references.
- `canon/characters/` — character facts, roles, relationships, and voice notes grounded in canon.
- `canon/locations/` — places, regions, and setting details from canon.
- `canon/plot-points/` — canonical events and sequencing.
- `canon/factions/` — nations, organizations, and allegiance notes.
- `canon/timeline/` — chronology across game/film/anime/DLC context.
- `canon/lore/` — world rules, mythology, artifacts, and systems.
- `canon/terms/` — glossary of recurring names and terminology.
