# Fanon Reference System (Flexible)

This directory stores **FFXV fanon**: community patterns, popular interpretations, and AU/headcanon conventions.

- `fanon/` is intentionally flexible and can evolve over time.
- Keep canon-verified facts in `canon/`.
- Keep project-specific notes in each story folder under `stories/`.

## Fanon structure
- `fanon/index.md` — master index of fanon references.
- `fanon/characters/` — fanon portrayals, common traits, and characterization patterns.
- `fanon/locations/` — fanon setting interpretations and recurring environment tropes.
- `fanon/plot-points/` — popular fic beats, divergence patterns, and fix-it structures.
- `fanon/factions/` — fanon readings of groups, alliances, and political dynamics.
- `fanon/timeline/` — common timeline rewrites and divergence anchors.
- `fanon/lore/` — fanon worldbuilding extensions and reinterpretations.
- `fanon/terms/` — glossary of fandom shorthand and recurring tags.
- `fanon/aus/` — alternate-universe frameworks and reusable premises.
